# Vehicle Rental System (C++ Project with Web Interface)

## Description
This is a **Vehicle Rental System** implemented in **C++** with a simple web interface.  
Users can book vehicles, view available vehicles, and manage rental operations using an interactive front-end.

## Features
- View available vehicles and their details
- Book a vehicle and update availability
- Store and retrieve data from a file-based database
- Interactive web interface using HTML, CSS, and JavaScript
- Object-Oriented C++ backend with structured data management

## Technologies Used
- **C++** for backend logic
- **HTML, CSS, JavaScript** for frontend interface
- File handling in C++ for data storage
- Optional: GitHub Pages for live demo (frontend only)
